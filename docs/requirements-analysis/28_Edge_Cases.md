# الحالات الاستثنائية — Edge Cases

> **رقم العنصر**: #28 | **المحور**: و | **الحالة**: مواصفات نهائية

**ملاحظات حاكمة**:
- VIN مكرَّر: **hard block** (ليس warning) على order_items نشطة بنفس VIN — خطأ validation (`VIN_DUPLICATE`).
- إلغاء فاتورة منفرد: يتم عبر Avoir لا عبر DELETE (راجع `07_Workflows.md` قسم 16).
- Concurrency على gift_pool: القفل يحدث **مرة واحدة في بداية الـ tx** لكل rows الـ pool المعنية (Report 1 H2 — راجع `29_Concurrency.md`).
