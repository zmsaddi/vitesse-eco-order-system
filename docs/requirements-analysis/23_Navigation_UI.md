## Sidebar (القائمة الجانبية)

- **الموقع**: يمين الشاشة (RTL)
- **Desktop (>1024px)**: ثابت، عرض 260px
- **Tablet/Mobile (<1024px)**: drawer منزلق (shadcn Sheet)
- **القوائم مخصصة حسب الدور** — كل دور يرى قوائم مختلفة (تفاصيل في 15_Roles_Permissions.md)

### عناصر Sidebar

| العنصر | الموقع |
|--------|--------|
| شعار + اسم التطبيق | أعلى |
| قوائم التنقل (مجمّعة) | وسط |
| معلومات المستخدم (اسم + دور) | أسفل |
| زر تسجيل الخروج | أسفل |

## Topbar (الشريط العلوي)

| العنصر | الموقع | ملاحظة |
|--------|--------|--------|
| Hamburger (موبايل) | يمين | يفتح Sidebar drawer |
| Breadcrumbs | وسط | مسار الصفحة الحالية |
| بحث (Ctrl+K) | — | يفتح Command Palette |
| جرس الإشعارات | يسار | عدد غير المقروء + dropdown |
| قائمة المستخدم | يسار | اسم + دور + تسجيل خروج |

## Command Palette (Ctrl+K)

- shadcn Command component
- بحث عام: عملاء، موردين، منتجات، طلبات
- تنقل سريع للصفحات
- فلترة حسب الدور
- لوحة مفاتيح: أسهم + Enter + Escape

## RTL

- `direction: rtl` على html
- خط Cairo (Google Fonts via next/font)
- حقول LTR: هاتف، إيميل، VIN، أرقام، username
- Sidebar يمين، محتوى يسار
