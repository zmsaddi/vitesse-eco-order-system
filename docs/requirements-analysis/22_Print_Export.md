# الطباعة والتصدير — Print & Export

> **رقم العنصر**: #22 | **المحور**: د | **الحالة**: قيد التحديث

---

## PDF الفاتورة

- **اللغة**: الفرنسية (متطلب قانوني)
- **المحتوى**: بيانات الشركة (SIRET, SIREN, APE) + بيانات العميل + أصناف متعددة + VIN لكل صنف
- **الهدايا**: تظهر بسعر البيع + خصم 100% = 0€
- **TVA**: تُحسب من TTC عند التوليد — لا تُخزّن (قرار H1/M2)
- **الحالات**: مؤكد / ملغي (حذف ناعم — BR-65)
- **الترقيم**: INV-YYYYMM-NNN (ذري)

## PDF الكتالوج

- **3 لغات**: العربية / الإنجليزية / الفرنسية (المستخدم يختار)
- **بدون أسعار** — صور + اسم + وصف + مواصفات
- **فلتر**: الكل أو المتاح في المخزون فقط
- **المنتجات المعطّلة** لا تظهر
- صفحة لكل منتج: صورة رئيسية + صور مصغرة

## CSV Export

- **متوفر في كل صفحة** تحتوي جدول (مدمج في DataTable)
- **التنسيق** (قرار M13):
  - فاصل: `;` (فاصلة منقوطة — لتوافق Excel الفرنسي)
  - أرقام عشرية: `.` (نقطة)
  - تواريخ: ISO `YYYY-MM-DD`
  - ترميز: UTF-8 مع BOM
- **الأعمدة**: حسب أعمدة الجدول المرئية (تحترم رؤية الدور)
