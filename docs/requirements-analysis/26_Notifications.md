# الإشعارات والتنبيهات — Notifications

> **رقم العنصر**: #26 | **المحور**: هـ | **الحالة**: قيد التحديث

---

## البنية التقنية

| البند | التفصيل |
|-------|---------|
| Real-time | SSE (Server-Sent Events) مع reconnect تلقائي |
| Fallback | polling كل 30 ثانية (Vercel Free timeout ~60s) |
| التخزين | جدول notifications في DB |
| التفضيلات | جدول notification_preferences (قابل للتكوين لكل مستخدم) |
| القنوات | in_app (حالياً) + email + push (مستقبلاً) |

## واجهة الإشعارات

- **جرس** في Topbar مع عدد غير المقروء (badge)
- **Dropdown** عند النقر: آخر الإشعارات + "عرض الكل"
- **صفحة كاملة** `/notifications`: كل الإشعارات + فلاتر

## تفضيلات المستخدم

كل مستخدم يمكنه تفعيل/تعطيل كل نوع إشعار لكل قناة (in_app / email / push).
جدول `notification_preferences`: (user_id, notification_type, channel, enabled).
