## طبقات الحماية

| # | الطبقة | التفصيل |
|---|--------|---------|
| 1 | **Neon PITR** | Point-in-time recovery مدمج (تلقائي) |
| 2 | **نسخ يدوي** | زر في الإعدادات لـ PM/GM — تنزيل pg_dump / استعادة (قرار C4) |
| 3 | **حذف ناعم** | لا حذف نهائي للطلبات/التوصيلات/الفواتير/المنتجات (H5) |
| 4 | **حماية Reset** | ALLOW_DB_RESET=false + NODE_ENV check + confirmation phrase |
| 5 | **SSL** | اتصال Neon يستخدم SSL تلقائياً |
| 6 | **سجل تدقيق** | activity_log + cancellations + price_history + treasury_movements |

## لا نسخ خارجي أتمتي

ليس أولوية حالياً. النسخ اليدوي (C4) + Neon PITR كافي لـ 10-20 مستخدم.
