## البحث الشامل (Global Search)

- **الاختصار**: Ctrl+K / Cmd+K
- **المكون**: GlobalSearch.js
- **التحميل**: كسول (lazy-load) عند أول فتح، تخزين مؤقت 2 دقيقة
- **الحد الأقصى**: 5 نتائج لكل فئة
- **التأخير**: 150ms debounce

### الكيانات المبحوثة حسب الدور

| الكيان | الحقول المبحوثة | الرابط | الأدوار |
|--------|----------------|--------|---------|
| العملاء | name, description_ar, phone, email, address | /clients/{id} | admin, manager |
| الموردين | name, phone | /suppliers | admin, manager |
| المنتجات | name, category | /stock | admin, manager, seller |
| المبيعات | client_name, product_name, notes | /sales | admin, manager, seller |
| المشتريات | supplier_name, product_name, notes | /purchases | admin, manager |
| المصاريف | description, category, notes | /expenses | admin, manager |

## الترتيب (Sorting)

- **الآلية**: الضغط على عنوان العمود
- **الافتراضي**: الأحدث أولًا (معظم الصفحات)
- **الاستثناء**: المخزون = الاسم تصاعدي، العملاء = الاسم تصاعدي
