# البحث والفلترة — Search & Filtering

> **رقم العنصر**: #21 | **المحور**: د | **الحالة**: قيد التحديث

---

## فلاتر كل صفحة

| الصفحة | الفلاتر |
|--------|--------|
| الطلبات | تاريخ من-إلى، اسم العميل، الحالة، حالة الدفع، البائع (PM/GM/Manager) |
| المشتريات | تاريخ من-إلى، بحث (مورد/منتج/كود)، المورد، حالة الدفع |
| المصاريف | تاريخ من-إلى، الفئة، بحث بالوصف |
| التوصيلات | تاريخ من-إلى، الحالة، السائق |
| مهام السائق | الحالة (بانتظار/جاري/مكتمل) |
| تحضير الطلبات | الحالة (قيد التحضير/جاهز) |
| الفواتير | بحث (عميل/كود/منتج/VIN)، الحالة |
| الصندوق | تاريخ من-إلى، الحساب، فئة الحركة |
| المخزون | بحث بالاسم، الفئة، الحالة (نشط/معطّل)، مستوى المخزون (متوفر/منخفض/نفذ)، علامة هدية |
| العملاء | بحث (اسم/هاتف)، فلتر ديون (مع دين/بدون) |
| الموردين | بحث (اسم/هاتف) |
| عمولتي | تاريخ من-إلى، الحالة (مصروف/غير مصروف) |
| سجل النشاطات | تاريخ، المستخدم، نوع الكيان، نوع الإجراء |
| Dashboard | فلتر تاريخ (الشهر/السنة/مخصص) |
| الجرد | تاريخ الجرد، فلتر الفروقات |

## بحث عام (Command Palette)

- **Ctrl+K** → shadcn Command
- يبحث في: عملاء، موردين، منتجات، طلبات، مشتريات
- نتائج مجمّعة حسب النوع
- تنقل بالأسهم + Enter
- فلترة حسب دور المستخدم

## ترتيب الجداول

نقر على عنوان العمود → ترتيب تصاعدي/تنازلي. سهم ▲▼ يظهر.
