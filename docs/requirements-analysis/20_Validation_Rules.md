## قواعد التحقق لكل حقل

### الحقول المشتركة

| الحقل | القاعدة | الرسالة |
|-------|---------|---------|
| التاريخ | مطلوب، صيغة YYYY-MM-DD | "التاريخ مطلوب" / "صيغة التاريخ غير صحيحة" |
| الكمية | رقم > 0 | "يجب أن يكون أكبر من 0" |
| السعر | رقم ≥ 0 | "يجب أن يكون رقماً" / "لا يمكن أن يكون سالباً" |
| الاسم | نص، 1+ حرف | "مطلوب" |

### المبيعات — تحقق خاص

| القاعدة | التفصيل | الرسالة |
|---------|---------|---------|
| سعر البيع ≥ buy_price | جميع الأدوار — حماية من الخسارة | (admin: يكشف التكلفة / seller: رسالة عامة) |
| سعر البيع ≥ sell_price | البائع فقط — حد السعر الموصى | "لا يمكن البيع بأقل من السعر الموصى" |
| الكمية ≤ المخزون | FOR UPDATE + فحص | "الكمية المطلوبة أكبر من المخزون المتاح" |
| كاش/بنك: dpe = total | الدفعة المقدمة = الإجمالي | "البيع النقدي/البنكي يتطلب دفع المبلغ بالكامل عند التوصيل" |
| آجل: 0 ≤ dpe ≤ total | الدفعة المقدمة ≤ الإجمالي | — |

### المشتريات — تحقق خاص

| القاعدة | التفصيل | الرسالة |
|---------|---------|---------|
| الفئة مطلوبة | لا يمكن ترك الفئة فارغة | "فئة المنتج مطلوبة" |
| sell_price ≥ unit_price | السعر الموصى ≥ سعر الشراء | "السعر الموصى لا يمكن أن يكون أقل من سعر الشراء" |
| paid ≤ total | المدفوع ≤ الإجمالي | — |

### التسويات — تحقق خاص

| القاعدة | التفصيل | الرسالة |
|---------|---------|---------|
| المبلغ ≤ الرصيد | لا يمكن صرف أكثر من المتاح | تحقق مزدوج (عند الإدخال + قبل الإرسال) |
| النوع صحيح | seller_payout أو driver_payout | "نوع التسوية غير صحيح" |

### توزيع الأرباح — تحقق خاص

| القاعدة | التفصيل | الرسالة |
|---------|---------|---------|
| 1+ مستلم | مصفوفة غير فارغة | "يجب اختيار مستلم واحد على الأقل" |
| النسب = 100% | مجموع النسب (± 0.01%) | — |
| المبلغ ≤ المتاح | لا يتجاوز صافي الربح | "المبلغ يتجاوز صافي الربح المتاح" |
| النسبة 0-100 | لكل مستلم | "النسبة بين 0 و 100" |

### المستخدمين

| القاعدة | التفصيل | الرسالة |
|---------|---------|---------|
| اسم المستخدم فريد | لا يتكرر | تحقق server-side |
| كلمة المرور 6+ | الحد الأدنى | — |
| الدور صحيح | admin/manager/seller/driver | — |

## تحقق قاعدة البيانات (CHECK Constraints)

| الجدول | العمود | القيد |
|--------|--------|-------|
| sales | payment_status | IN ('pending','partial','paid','cancelled') |
| payments | type | IN ('collection','refund','advance') |
| payments | payment_method | IN ('كاش','بنك') |
| cancellations | invoice_mode | IN ('soft','delete') |
