# متطلبات الطباعة والتصدير — Print / PDF / Export Requirements

> **رقم العنصر**: #21 | **المحور**: د | **الحالة**: مكتمل

---

## 1. فاتورة PDF

- **المسار**: GET /api/invoices/[id]/pdf
- **التوليد**: يُفتح في تبويب جديد للطباعة
- **المحتوى**:
  - بيانات الشركة (من settings): اسم، شكل قانوني، SIREN، SIRET، APE، عنوان
  - رقم الفاتورة (INV-YYYYMM-NNN)
  - تاريخ الفاتورة
  - بيانات العميل: اسم، هاتف، إيميل، عنوان
  - تفاصيل المنتج: اسم، كمية، سعر الوحدة، الإجمالي
  - طريقة الدفع
  - VIN (للدراجات/السكوتر)
  - اسم البائع + اسم السائق
  - نسبة TVA + رقم TVA

---

## 2. تصدير CSV (لوحة التحكم)

- **المتاح لـ**: admin, manager فقط
- **المحتوى**:
  - 3 طرق P&L (Pipeline, Accrual, Cash)
  - جدول التدفق النقدي (كاش + بنك)
  - الرؤوس: البند (item)، المبلغ (amount)
- **اسم الملف**: `vitesse-eco-report-{dateFrom}_{dateTo}.csv`

---

## 3. لا يوجد تصدير آخر حاليًا

- لا تصدير Excel
- لا طباعة سند توصيل مستقل
- لا تصدير قائمة العملاء أو الموردين
